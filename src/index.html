HTML5 + Scss + Gulp Boilerplate

This is the simplest and powerful boilerplate to create your projects. Just install and run.

1. Installation

Clone repository to your project folder:

git clone git@github.com:toxuh/html5-scss-bolerplate.git .

Then install packages:

npm i // Node
yarn // Yarn

2. Usage

To run project in development mode, run:

npm run start // Node
yarn start // Yarn

Code will be served on 3000 port. It has autoreload, so no more F5 spam.

To build the project, run:

npm run build // Node
yarn build // Yarn

3. Details

Boilerplate supports scss, html partials
